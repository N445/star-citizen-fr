# Comment installer la traduction
## Automatique
Télécharger le fichier `download_trad.bat` et déplacé le dans le dossier de jeux `Roberts Space Industries\StarCitizen\LIVE` ou `Roberts Space Industries\StarCitizen\PTU`
Pour récupérer le traduction double cliquez dessus, ça automatiquement récupérer la traduction LIVE OU PTU

## Manuel
Copier le fichier  `user.cfg` a la racine du jeu `Roberts Space Industries\StarCitizen\LIVE`
Copier le dossier `data` a la racine du jeu `Roberts Space Industries\StarCitizen\LIVE`
